# Android Homebrew Tap

This repository contains Homebrew (aka, Brew) installable packages ("formulae")
for Android developer tools.

## How to install packages from here?

```
brew install android/tap/<package_name>
```

You can also add the tap to make packages available in `brew search` results:

```
brew tap android/tap
```

## What packages are available?

```
# Casks
brew install android/tap/android-cli
```
